## ❓ Troubleshooting

### "Port 8000 is already in use"
- **Solution 1:** Close other applications using port 8000
- **Solution 2:** Wait 30 seconds and try again
- **Solution 3:** Edit `serve.py` and change `PORT = 8000` to `PORT = 8001`

### "Python is not installed"
- **Windows:** Download from https://www.python.org/downloads/
- **macOS:** Run `brew install python3`
- **Linux:** Run `sudo apt install python3 python3-pip`

### Browser shows 404 error
- Make sure you're in the correct directory with all files
- Check that `index.html`, `styles.css`, and `script.js` are present

### Website looks broken/incomplete
- Clear your browser cache (Ctrl+Shift+Delete)
- Try a different browser (Chrome, Firefox, Safari, Edge)
- Make sure JavaScript is enabled
